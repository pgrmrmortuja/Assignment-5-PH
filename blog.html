What is the DOM in JavaScript?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, which developers can interact with using JavaScript to manipulate the content, structure, and style of web pages dynamically.

How do you find an element in the DOM?

You can select an element from the DOM using various methods like document.getElementById(), document.querySelector(), or document.querySelectorAll(). These methods allow you to target specific elements by their ID, class, or other attributes.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element instead of multiple child elements. It takes advantage of event bubbling, where events triggered on child elements propagate up to the parent. This is useful for improving performance, especially when dynamically adding or removing elements.
